汇报页配图 / 视频放置说明
========================

图片与视频资源请放在本目录（public/report_media）。报告类文档请放在 public/report。网页会直接读取本目录下的文件，支持 .webp（推荐）、.jpg、.png、.gif、.mp4、.webm。

一、当前页面期望的文件名（三处配图）
------------------------------------
页面默认按 .webp 读取，请将您的资源命名为：

1. 设计创新（第 06 节）  →  design_media.webp（或 .jpg / .png / .gif / .mp4）
2. 应用场景（第 07 节）  →  scenarios_media.webp（或同上）
3. 市场反响（第 08 节）  →  market_media.webp（或同上）

若某节是视频，请命名为 design_media.mp4 等，并告诉我「该节用视频」，我会改为 <video> 展示。

二、其他后缀
------------
若您用的是 .jpg / .png / .gif，告诉我对应节和后缀，我会把该节的 extension 改过去。.webp 可直接在网页中显示，无需转换。

三、report 文件夹
-----------------
汇报相关报告、提纲等已放在 public/report，供查阅或导出；网页配图只从本目录（report_media）读取。

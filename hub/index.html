新报告流程

以后不要再为每个汇报创建单独仓库或桌面启动器。先用脚本创建报告目录，完成后 build，最后用归档脚本压缩成汇报留档。

npm run new-report -- <id> <标题>
npm run check
npm run build -- <report-id>
npm run archive -- <report-id>

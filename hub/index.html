新报告流程

固定流程可以减少目录混乱：先创建报告目录，再补素材和页面，完成后构建并检查。不要为单份报告再创建仓库或桌面启动器。

npm run new-report -- <id> <标题>
npm run check
npm run build -- <report-id>
npm run archive -- <report-id>

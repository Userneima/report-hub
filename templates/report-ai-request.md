# 新报告 AI 请求模板

把下面这段发给 AI，可以直接进入统一工作流。

```text
请在 report-hub 中制作一份新的网页形式汇报 PPT。

主题：
【在这里写主题】

使用方式：
- 不要新建 GitHub 仓库。
- 不要新建桌面启动器。
- 使用现有 report-hub 结构。
- 如果还没有报告目录，先运行 npm run new-report -- <id> <标题>。

内容流程：
- 先写 0.brief/brief.md，明确汇报目标、受众、评分点、叙事模块和页面规划。
- 再把初步调研、AI 填充、参考链接、待核验事实写入 1.research/notes.md。
- 页面内容薄弱时，需要主动联网补充事实、案例、对比、图表或风险边界。
- 每页必须有明确观点，中文字号要适合教室投影。

素材流程：
- 我手动下载的图片、视频、PDF 或截图会放在 2.assets/raw/。
- 你需要筛选、重命名并复制可用素材到 2.assets/selected/。
- 页面最终引用的素材放入 public/media/。
- 外部素材必须记录在 2.assets/sources.md：文件、来源 URL、用途、状态、备注。

验收：
- 完成后运行 npm run audit 和 npm run check。
- 打开报告中心或报告页面做运行时检查。
- 最后告诉我改了什么、验证结果、还有哪些风险。
```
